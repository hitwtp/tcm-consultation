# TCM Consultation - 中医问诊系统

基于 Electron + React 的中医智能问诊软件，实现望闻问切四诊合参的辅助诊断功能。

## 功能特性

### 🏥 四诊信息采集
- **望诊**: 通过摄像头采集舌象、面色、手部图像
- **闻诊**: 记录患者声音特征描述
- **问诊**: 症状选择与主诉录入
- **切诊**: 脉象类型选择与描述

### 📊 智能诊断
- 基于症状匹配的中医辨证分析
- 8种常见证型识别
- 详细的病机分析与治疗原则

### 💊 处方用药
- 经典方剂推荐
- 中药配伍建议
- 用法用量说明

### 📋 报表导出
- 一键生成 PDF 诊断报告
- 包含患者信息、诊断结果、处方建议

## 技术栈

- **前端框架**: React 18
- **桌面框架**: Electron 25
- **构建工具**: Vite
- **PDF生成**: jsPDF
- **语言**: TypeScript

## 安装运行

```bash
# 安装依赖
cd tcm-consultation
npm install

# 开发模式
npm run electron:dev

# 构建应用
npm run electron:build
```

## 项目结构

```
tcm-consultation/

│   ├── App.tsx       # 主应用组件
│   ├── diagnosis.ts  # 诊断逻辑
│   ├── reportGenerator.ts  # PDF生成

```

## 证型数据库

系统支持以下中医证型的智能诊断：

| 证型 | 主要症状 |
|------|----------|
| 肝郁气滞 | 胁痛、抑郁、胀痛 |
| 心脾两虚 | 心悸、失眠、多梦 |
| 肾阳虚 | 畏寒、腰膝酸软、夜尿 |
| 脾胃湿热 | 口苦、脘腹胀满 |
| 肺气虚 | 咳嗽、气短、易感冒 |
| 痰湿体质 | 肥胖、痰多、胸闷 |
| 阴虚火旺 | 口干、潮热、盗汗 |
| 气血两虚 | 面色苍白、乏力 |
